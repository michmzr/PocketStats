# PocketStats

App for visualising [GetPocket](https://getpocket.com/) usage stats

## Config

### Backend

#### Environment variabled

- POCKET_CONSUMER_KEY
- MONGODB_HOST
- MONGODB_PORT
- MONGODB_USER
- MONGODB_PASS
- MONGODB_DB
- MONGODB_URI
